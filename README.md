# Password Checker (C# WinForms)

Приложение на C# WinForms для автоматической оценки надежности пароля.

Описание

После ввода первого символа приложение сразу начинает показывать уровень надежности пароля. Пользователь может продолжать вводить пароль, удалять символы и вводить новый пароль в любое время. Оценка обновляется автоматически по мере изменения текста.

Возможности

* автоматическая проверка надежности пароля при вводе;
* мгновенное обновление результата после каждого введенного символа;
* возможность дописывать пароль для повышения надежности;
* возможность удалить введенный текст и ввести другой пароль;
* простой графический интерфейс WinForms.

Требования

Для запуска проекта на компьютере необходимо:

* операционная система Windows;
* Visual Studio 2019, 2022 или новее;
* установленная рабочая нагрузка ".NET desktop development";
* .NET SDK или .NET Framework, в зависимости от версии проекта.

Как установить Visual Studio, если ее нет

1. Откройте официальный сайт Visual Studio.
2. Скачайте Visual Studio Community — это бесплатная версия.
3. Запустите установщик.
4. На этапе выбора компонентов обязательно отметьте рабочую нагрузку ".NET desktop development".
5. Дождитесь завершения установки.

Если приложение написано на старом .NET Framework, также проверьте, что установлены нужные компоненты для Windows Forms.

Как открыть проект

1. Скачайте архив с проектом или клонируйте репозиторий через Git.
2. Откройте папку проекта на компьютере.
3. Найдите файл решения с расширением `.sln`.
4. Дважды нажмите на этот файл.
5. Проект откроется в Visual Studio.

Если Visual Studio спрашивает, чем открыть файл, выберите Visual Studio.

Если Visual Studio есть, но WinForms не установлен

Если при открытии проекта Visual Studio сообщает, что не хватает компонентов, сделайте следующее:

1. Откройте Visual Studio Installer.
2. Найдите установленную версию Visual Studio.
3. Нажмите кнопку Modify или Изменить.
4. Установите рабочую нагрузку ".NET desktop development".
5. Дождитесь окончания установки и снова откройте проект.

Как запустить приложение

1. Откройте файл `.sln`.
2. Дождитесь, пока Visual Studio восстановит зависимости, если это потребуется.
3. Убедитесь, что выбран правильный стартовый проект.
4. Нажмите кнопку Start в Visual Studio или клавишу F5.
5. После сборки откроется окно приложения.

Если программа не запускается, попробуйте сначала выполнить Build -> Rebuild Solution.

Как пользоваться приложением

1. Запустите приложение.
2. Начните вводить пароль в поле ввода.
3. Уже после первого символа приложение покажет предварительную оценку надежности.
4. Продолжайте вводить символы, чтобы усилить пароль.
5. При необходимости удалите часть пароля или полностью очистите поле и введите новый вариант.
6. Наблюдайте, как меняется индикатор или текст с оценкой надежности.

Возможные проблемы и решения

Проект не открывается

Проверьте, что у вас установлена Visual Studio и открывается именно файл `.sln`.

Не хватает шаблона WinForms

Это означает, что не установлена нужная рабочая нагрузка. Откройте Visual Studio Installer и добавьте ".NET desktop development".

 Ошибка при сборке

Попробуйте следующие действия:

1. Build -> Clean Solution.
2. Build -> Rebuild Solution.
3. Закройте и снова откройте Visual Studio.

Приложение не запускается

Проверьте:

* выбран ли правильный startup project;
* нет ли ошибок в коде;
* установлены ли нужные компоненты .NET.

Структура проекта

Обычно проект WinForms содержит такие файлы:

* `Program.cs` — точка входа в программу;
* `Form1.cs` — основная логика формы;
* `Form1.Designer.cs` — автоматически сгенерированный код интерфейса;
* `.csproj` — файл проекта;
* `.sln` — файл решения Visual Studio.
